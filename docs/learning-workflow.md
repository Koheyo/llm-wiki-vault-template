# Course Learning Workflow

The template includes a reusable study workflow that turns ingested material into durable learning notes.

## Phases

1. Identify material and target course/project.
2. Ingest source material while preserving raw/source context.
3. Create a lecture/source summary.
4. Promote reusable ideas into concept pages.
5. Ask the user to choose a learning mode.
6. Generate guided walkthrough, review pack, cheat sheet, or quiz.
7. Verify frontmatter, wikilinks, image embeds, and log the operation.

## Learning modes

- **Live slide tutor**: interactive slide-by-slide tutoring.
- **Auto detailed walkthrough**: complete lecture-style explanation.
- **Key-slide walkthrough**: focused review of high-value slides.
- **Exam review**: formulas, pitfalls, drills, and quiz.
- **Algorithm-first**: reorganize slides by concepts/algorithms.

## Principle

Ingest prepares assets. Teaching starts only after the user explicitly chooses a learning mode.
