# Example Source Summary

This page demonstrates how a raw source is converted into a concise maintained summary.

## Main idea

A source summary should capture the source's durable ideas, not every detail. Link key terms to concept pages such as [[wiki/concepts/retrieval-augmented-generation]] and contextual entities such as [[wiki/entities/example-course]].

## Key points

- Keep source material in `raw/`.
- Write durable synthesis in `wiki/`.
- Promote reusable ideas into concept pages.
- Link summaries, entities, and concepts together.

## Follow-ups

- Create more concept pages for ideas that recur across sources.
- Mark this page `curated` after human review.
