# Retrieval-Augmented Generation

Retrieval-augmented generation is a pattern where a language model uses retrieved external context before producing an answer.

## Why it matters

RAG can reduce reliance on model memory, make answers easier to trace to sources, and support domain-specific knowledge without retraining the model.
